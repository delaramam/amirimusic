# Delaram Amiri - Official Website

Personal website for Delaram Amiri, Persian tombak player and classical musician.

**Live site:** [amirimusic.com](https://amirimusic.com)

## Pages

- **Home** - Introduction, featured concert, newsletter signup
- **Bio** - Biography and photo gallery
- **Events** - Upcoming and past performances
- **Gallery** - Video performances
- **Faraz Ensemble** - Ensemble information and members
