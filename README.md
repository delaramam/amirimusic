# Delaram Amiri - Persian Classical Music Website

This is a website for Persian classical music artist Delaram Amiri, featuring her performances, biography, and upcoming concerts.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Modern Layout**: Clean, elegant design inspired by classical music aesthetics
- **Interactive Navigation**: Smooth scrolling and mobile-friendly menu
- **Concert Information**: Featured upcoming performances and events
- **Newsletter Signup**: Email collection for performance updates
- **Gallery**: Photo showcase of performances
- **Ensemble Information**: Details about the Faraz Ensemble

## File Structure

```
amiri-music-website/
├── index.html          # Main HTML file
├── styles.css          # All CSS styles
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Setup Instructions

1. **Download the files** to your computer
2. **Open `index.html`** in any modern web browser
3. **For GitHub Pages deployment**:
   - Create a new repository on GitHub
   - Upload all files to the repository
   - Go to Settings → Pages
   - Select "Deploy from a branch" → "main"
   - Your site will be available at `https://yourusername.github.io/repository-name`

## Customization

### Adding Real Images
Replace the placeholder divs with actual images:

```html
<!-- Replace this: -->
<div class="image-placeholder">Featured Image</div>

<!-- With this: -->
<img src="path/to/your/image.jpg" alt="Description">
```

### Updating Content
- Edit text in `index.html`
- Modify colors in `styles.css` (search for `#8B4513` to change the main color)
- Add more sections by copying existing HTML structure

### Adding Real Functionality
- Replace email form with actual email service (Mailchimp, ConvertKit, etc.)
- Add real social media links
- Connect ticket links to actual ticketing systems
